#  Company Sales & Profit Dashboard - Power BI

This Power BI dashboard analyzes company sales, profit, yearly trends, and product performance using interactive visualizations.

---

##  Dashboard Features
- Sales Analysis
- Profit Analysis
- Product-wise Sales
- Vendor Analysis
- Country-wise Sales Map
- Yearly Sales Trend
- Interactive Filters and Slicers

---

##  Tools Used
- Power BI
- Excel / CSV Dataset
- Data Visualization
- DAX
